入门

🚀 快速开始

5 分钟完成环境配置，启动你的第一个 AI 助手

1 环境要求

✓

Node.js 18+ (推荐 v20+)

OpenClaw 基于 Node.js 开发，需要较新版本支持

✓

npm 或 pnpm 包管理器

用于安装 OpenClaw 核心依赖

✓

Git (可选但推荐)

用于版本控制和同步配置

# 检查 Node.js 版本

node -v

2 安装 Node.js

macOS / Linux (推荐 nvm)

# 安装 nvm (Node 版本管理器)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装最新 LTS 版本
nvm install --lts
nvm use --lts

Windows

下载安装器：https://nodejs.org/

选择 LTS (长期支持) 版本，一路下一步即可

3 安装并启动 OpenClaw

# 使用 npx 一键启动 (无需全局安装)

npx openclaw start

💡 提示

首次运行会自动创建 workspace 目录 (~/.openclaw/workspace)，并初始化配置文件。 后续可以直接运行 openclaw 命令。

4 配置本地 Ollama 大模型

OpenClaw 支持接入本地运行的 Ollama 大模型，无需依赖云端 API，数据完全本地化。

4.1 安装 Ollama

# macOS / Linux
curl -fsSL https://ollama.com/install.sh | sh

# Windows: 访问 https://ollama.com 下载安装包

4.2 下载模型

# 推荐模型 (根据显存选择)
ollama pull qwen2.5:7b # 7B 参数，约 4GB 显存
ollama pull qwen2.5:14b # 14B 参数，约 8GB 显存
ollama pull qwen3-max # 最新版本 (如有)

4.3 配置 OpenClaw 连接 Ollama

编辑 workspace 中的配置文件，添加 Ollama 提供商：

# ~/.openclaw/workspace/config.json (示例)
{
"providers": {
"ollama": {
"baseUrl": "http://localhost:11434",
"model": "qwen2.5:7b"
}
}
}

✅ 验证安装

启动后，在聊天中输入以下命令测试：

/openclaw status
/help

如果看到正常响应，恭喜你已经成功入门！
